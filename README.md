# Search function prefixes
Can be found and changed on search.js
  
  -im   DuckDuckGo image search
  
  -yt   YouTube search
  
  -wk   Wikipedia search
  
  -wf   Wolfram Alpha input
  
  -tv   TVTropes search
  
  -r/   enter subreddit
  
  
  
  no prefix defaults to DuckDuckGo
